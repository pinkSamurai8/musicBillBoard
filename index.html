About

Hi! I'm Faith, I study computer science with anticipation to become a software designer. I designed this site as a fun project. I've always been fascinated with zodiac signs (well before realizing how deep the zodiac rabbit hole goes, of course), and who doesn't love music? I thought it would be fun to create a place where you could type in your birthday and be paired with a top-charting musician. This was my first time using JavaScript, but I was able to apply some of the knowledge I gained from learning C++ and Java. The fundamentals are so important, I can see how they're applicable from language to language. Hope you had some fun exploring! Thanks for visiting.
